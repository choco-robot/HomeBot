# HomeBot 人体跟随功能使用指南

## 功能简介

人体跟随功能基于 **YOLO26** 模型实现实时人体检测，结合目标跟踪和视觉伺服控制算法，实现机器人自动跟随目标人体。

## 技术特性

| 特性 | 说明 |
|------|------|
| **检测模型** | YOLO26-nano (~2.4MB) |
| **检测速度** | CPU 38ms/帧，比YOLO11快43% |
| **跟踪算法** | IoU匹配 + 卡尔曼滤波 |
| **控制算法** | 视觉伺服 (Visual Servoing) |
| **通信协议** | ZeroMQ REQ-REP |
| **优先级** | auto=3（可被emergency抢占）|

## 配置说明

配置文件位于 `software/src/configs/config.py`：

```python
@dataclass
class HumanFollowConfig:
    # 模型配置
    model_path: str = "models/yolo26n.pt"
    conf_threshold: float = 0.5
    
    # 跟踪配置
    max_tracking_age: int = 30          # 最大丢失帧数
    min_iou_threshold: float = 0.3      # IoU匹配阈值
    target_selection: str = "center"    # 选择策略: center/largest/closest
    
    # 推理优化
    inference_size: int = 320           # 输入分辨率
    use_half_precision: bool = False    # FP16半精度
    
    # 跟随控制
    target_distance: float = 1.0        # 目标距离（米）
    kp_linear: float = 0.001            # 线速度P系数
    kp_angular: float = 0.003           # 角速度P系数
    max_linear_speed: float = 0.3       # 最大线速度
    max_angular_speed: float = 0.8      # 最大角速度
    dead_zone_x: int = 50               # 水平死区（像素）
    
    # 安全配置
    stop_on_lost: bool = True           # 丢失目标时停止
    search_on_lost: bool = True         # 丢失时搜索
```

## 控制逻辑

### 水平控制（旋转）
- 目标在画面左侧 → 左转
- 目标在画面右侧 → 右转
- 死区内 → 不旋转

### 距离控制（前后）
- 目标太小（太远）→ 前进
- 目标太大（太近）→ 后退
- 在目标面积范围内 → 停止

### 目标丢失处理
1. **暂时丢失**（<60帧）：原地旋转搜索
2. **完全丢失**（>60帧）：停止移动，进入空闲状态

## 可视化信息

调试窗口显示以下信息：
- **绿色细框**: 所有检测到的人体
- **红色粗框**: 当前跟踪的目标
- **黄色圆点**: 目标中心
- **蓝色连线**: 目标与画面中心的偏移
- **绿色十字**: 画面中心
- **信息面板**: 模式、FPS、速度、目标数量

## 安全机制

1. **速度限制**：最大线速度和角速度可配置
2. **死区控制**：避免抖动
3. **丢失保护**：目标丢失自动停止或搜索
4. **急停响应**：可被emergency优先级抢占
5. **通信超时**：底盘通信超时自动停止

## 常见问题

### Q: 模型下载失败怎么办？
A: 手动下载：
```bash
wget https://github.com/ultralytics/assets/releases/download/v8.3.0/yolo26n.pt -O software/models/yolo26n.pt
```

### Q: 检测不到人体？
A: 检查：
1. 摄像头是否正常工作
2. 置信度阈值是否过高（默认0.5）
3. 光线是否充足

### Q: 跟随不流畅？
A: 调整参数：
- 降低 `inference_size` 到 320
- 调整 `kp_linear` 和 `kp_angular`
- 增大 `dead_zone_x`

### Q: 如何只跟随特定的人？
A: 当前版本选择策略：
- `center`: 选择画面中央的人
- `largest`: 选择最大的人（最近）
- `closest`: 同largest

未来版本将支持人脸识别指定目标。
