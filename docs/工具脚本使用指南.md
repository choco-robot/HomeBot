## 设备信息查看工具

**脚本**: `list_devices.py`

用于快速查看系统中可用的串口、摄像头和麦克风设备，帮助用户确认设备端口号并修改配置。

### 基本用法

```bash
cd software
python tools/list_devices.py
```

### 功能特性

| 设备类型 | 说明 |
|---------|------|
| 📟 串口设备 | 列出所有COM端口、设备描述、VID/PID信息 |
| 📷 摄像头设备 | 列出可用摄像头索引、分辨率、帧率 |
| 🎤 麦克风设备 | 列出麦克风索引、名称、默认采样率 |

### 高级选项

```bash
# 测试摄像头（显示实时预览，按 'q' 退出）
python tools/list_devices.py --test-camera

# 测试麦克风录音（3秒）
python tools/list_devices.py --test-mic

# 测试指定麦克风
python tools/list_devices.py --test-mic --mic-index 1 --mic-duration 5
```

### 输出示例

```
============================================================
 📟 串口设备 (Serial Ports)
============================================================

   📟 串口: COM15
      描述: USB-SERIAL CH340 (COM15)
      硬件ID: USB VID:PID=1A86:7523
      VID/PID: 1A86:7523

============================================================
 配置摘要
============================================================

📟 串口配置 (ChassisConfig / ArmConfig):
   当前配置: chassis.serial_port = "COM15"
   当前配置: arm.serial_port = "COM15"

📷 摄像头配置 (CameraConfig):
   当前配置: camera.device_id = 0

🎤 麦克风配置 (SpeechConfig):
   当前配置: speech.mic_index = 1
```

工具会自动显示当前配置值，并推荐可用的设备，方便用户直接修改 `software/src/configs/config.py`。

## 机械臂校准工具

**脚本**: `calibrate_arm.py`

用于校准机械臂各关节的零点位置，将当前机械位置设为舵机中点（2048）。

> ⚠️ **注意**: 校准会立即生效并保存到舵机 EEPROM，校准前请确保机械臂处于正确的机械零点位置。

### 基本用法

```bash
cd software/src
python -m tools.calibrate_arm
```

### 交互式校准（推荐）

不带参数运行会进入交互式菜单：

```
============================================================
交互式机械臂校准
============================================================

关节列表:
  1: base (基座旋转)
  2: shoulder (肩关节)
  3: elbow (肘关节)
  4: wrist_flex (腕关节屈伸)
  5: wrist_roll (腕关节旋转)
  6: gripper (夹爪)
  0: 校准所有关节
  q: 退出
```

### 快速校准

```bash
# 快速校准所有关节（无需确认）
python -m tools.calibrate_arm --quick

# 只校准特定关节
python -m tools.calibrate_arm --quick --joints 1 2

# 只查看当前位置
python -m tools.calibrate_arm --verify

# 指定串口
python -m tools.calibrate_arm --port COM3 --quick
```

### 校准原理

飞特舵机的零点校准原理：
1. 失能舵机扭矩
2. 向寄存器40写入128，当前位置会被设为2048
3. 重新使能扭矩
4. 验证位置并设置最大/最小角度限制

## 机械臂调试工具

**脚本**: `debug_arm.py`

用于快速调试机械臂，包括一键失能扭矩、一键复位到休息位置、查看当前状态等功能。

### 基本用法

```bash
cd software/src
python -m tools.debug_arm
```

### 功能选项

```bash
# 一键失能扭矩（可手动调整机械臂位置）
python -m tools.debug_arm --disable

# 一键复位到休息位置
python -m tools.debug_arm --reset

# 查看当前状态
python -m tools.debug_arm --status

# 指定串口和速度参数复位
python -m tools.debug_arm --reset --port COM3 --speed 600 --acc 30
```

### 交互式菜单

不带参数运行会显示菜单：

```
============================================================
机械臂调试工具 - 交互式菜单
============================================================

功能选项:
  1. 一键失能扭矩 (可手动调整位置)
  2. 一键复位机械臂
  3. 查看当前状态
  4. 使能扭矩
  q. 退出
```

### 复位位置

复位时会将机械臂移动到配置文件中定义的 `rest_position`：

```python
# software/src/configs/config.py
rest_position = {
    "base": -90,        # J1: 基座旋转
    "shoulder": 0,      # J2: 肩关节
    "elbow": 150,       # J3: 肘关节
    "wrist_flex": 30,   # J4: 腕关节屈伸
    "wrist_roll": -90,  # J5: 腕关节旋转
    "gripper": 45,      # J6: 夹爪（半开）
}
```

## 模型下载工具

### YOLO 模型下载

**脚本**: `download_models.py`

下载人体跟随所需的 YOLO 检测模型。

```bash
cd software
python tools/download_models.py
```
默认下载到 `software/models/` 目录。
将下载的yolo26n.pt 转换为 ONNX 格式：

python tools/export_onnx.py models/yolo26n.pt

## 故障排查

### 串口权限问题（Linux）

```bash
# 添加用户到 dialout 组
sudo usermod -a -G dialout $USER
# 重新登录后生效
```

### 摄像头无法打开

1. 使用 `list_devices.py --test-camera` 测试
2. 检查是否有其他程序占用摄像头
3. 尝试更换摄像头索引（修改 `camera.device_id`）

### 麦克风无声

1. 使用 `list_devices.py --test-mic` 测试录音
2. 检查系统默认录音设备设置
3. 尝试更换麦克风索引（修改 `speech.mic_index`）
